About

Hello! I'm Amy from South Korea and have been in the U.S. for 9 years now. I have practiced fine art and graphic design in Idyllwild Arts Academy and School of the Art Institute of Chicago (SAIC) and then have transferred the University of California (USC) to study economics, computer programming, and web development.

The Nutritionify project was an application for hackathon: Athenahacks 2024. Nutritionify utilizes the Spotify API to draw real user data into the server, reports back the top tracks and info in the format of a nutrition guide. I developed the frontend components using Vite, JS, React, and Spotify API. We faced some challenges as it was our first time using Vite, Ngrok, APIs, but we learned a lot in the process.

The Sushi Shopping Cart is still under development. I initially created the responsive page with Javascript and HTML, then later converted it to React and used StackBlitz to deploy it. Similarly, the React Image Gallery was created using React and bootstrap. The Resident Evil 4 mock advertisement and Magazine showcase design and color choices following design principles and rationales.

My programming skills are still developing, and I'm eager to incorporate what I've learned and practiced in contemporary arts to improve design and function in solving problems.
